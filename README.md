# ĐỒ ÁN: WEBSITE "HÔM NAY ĂN GÌ ĐÂY"

Website hỗ trợ người dùng quyết định thực đơn mỗi ngày thông qua vòng quay ngẫu nhiên và tra cứu công thức nấu ăn.

## 🛠 Công nghệ sử dụng
- HTML5, CSS3, JavaScript (ES6+)

## 🚀 Các tính năng chính
- Vòng quay may mắn quyết định món ăn.
- Quản lý danh sách món ăn cá nhân (Thêm/Xóa).
- Tra cứu công thức nấu ăn chi tiết.
- Hệ thống ghi nhớ lịch sử và danh sách yêu thích (tích hợp LocalStorage).
